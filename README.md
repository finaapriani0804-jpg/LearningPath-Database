# LearningPath Database

Academic database project developed using Oracle Database and Oracle APEX.

## Features
- Database Design
- ERD (Barker Notation)
- CREATE TABLE
- PRIMARY KEY & FOREIGN KEY
- Trigger
- CRUD (Insert, Update, Delete)
- SQL Query Testing
  - JOIN
  - Self Join
  - Outer Join
  - Top-N Analysis
  - Substitution Variable

## Tools
- Oracle Database
- Oracle APEX
- SQL
